<summary><b>思路</b></summary>

- 使用哈希表保存已经出现过的数字及其下标；
- 利用差值可以在一次遍历中完成；

<details><summary><b>Python</b></summary>

```python
class Solution:
    def twoSum(self , numbers: List[int], target: int) -> List[int]:
        
        dt = dict()
        for i, x in enumerate(numbers):
            if (r := target - x) in dt:
                return [dt[r], i + 1]
            else:
                dt[x] = i + 1
        
        return []
```
