<summary><b>思路</b></summary>

- 利用 Python 内置的 `zip` 函数每次纵向取所有字符串的第 `i` 个字符；
- 对这些字符 `set` 后，如果都相同，则加入前缀，否则退出循环，返回结果；

<details><summary><b>Python</b></summary>

```python
class Solution:
    def longestCommonPrefix(self , strs: List[str]) -> str:
        
        ret = []
        for it in zip(*strs):
            if len(set(it)) == 1:
                ret.append(it[0])
            else:
                break
        
        return ''.join(ret)
```
